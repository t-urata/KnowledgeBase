TimeTracker FX 3.7 についてご説明します。

 

本バージョンの概要

TimeTracker FX 3.6に対し機能追加・改善をしたマイナーバージョンアップ版です。
Webアプリケーション版のタイムシート「Web Edition」を新規に搭載し、ブラウザからの工数入力を実現しました。

 

バージョンアップの対象となるエディション

Standard Edition および Professional Edition

 

新たに追加・改善された機能

概要については、こちらをご覧ください。

各機能の詳細については、こちらをご覧ください。

 

修正された不具合

本バージョンでは、以下の不具合を修正しました。

不具合の対象

不具合内容

修正後の動作

オートメーションAPI
（プランナー）

プロジェクト内にある100個程度のリンクの設定をまとめて更新すると、システムエラーが発生する場合がある。
(ガントチャートが表示された状態の場合のみ)

システムエラーが発生せず、正常に更新できるようになりました。

UpdateLinkメソッドでリンクを変更した場合、変更されたリンクの警告アイコンが更新されない。
(その後WBSを編集すると更新される)

リンクを更新後、警告アイコンの表示がすぐに正常に表示するようになりました。

アナリスト

リソース負荷ビューで、プロジェクト期間外の計画工数が集計されない。
(実績工数が存在しないプロジェクトのみ)

プロジェクト期間外の計画工数もリソース負荷ビューで集計されるようになりました。

Web API
(タイムシート)

アカウントの実績工数を取得しようとすると、システムエラーが発生して取得できない場合がある。
(終了していない無効なプロジェクトのタスクに割り当てがある場合のみ)

システムエラーが発生せず、実績が取得できるようになりました。

【TimeTracker FX 3.6をお使いの場合】

バージョンアップについて

    • TimeTracker FX 3.7へバージョンアップする必要はありますか？

      本バージョンでは、ユーザ様の操作性向上のために機能追加や改善を行っております。

      可能な限りバージョンアップすることを推奨します。

       

    • TimeTracker FX 3.7を入手した後は、どうすればよいですか？

      ダウンロードファイルを展開した中または、CDの中にあるautorun.exeを実行してください。

       

    • アップグレードはサーバーマシンとクライアントPCそれぞれでどのように行えばよいですか？

      サーバーマシンは、データベースにTimeTracker FX 3.7のライセンスキーを登録するだけです。
      クライアントPCはTimeTracker FX3.6のままでも利用できます。
      TimeTracker FX 3.7の機能を利用する場合はクライアントPCもアップグレードしてください。

       

    • TimeTracker FX 3.6で利用していたライセンスキーをTimeTracker FX 3.7で利用できますか？

      利用できません。

      別途ご提供するTimeTracker FX 3.7のライセンスキーをご利用ください。

       

    • TimeTracker FX 3.6で利用していたデータベースを引き続き利用できますか？

      TimeTracker FX 3.6で利用していたデータベースは、ライセンス情報の更新のみで、そのままご利用いただけます。

       

    • TimeTracker FX 3.6のオートメーションAPIで作成したアプリケーションは利用できますか？

      利用できます。

       

    • TimeTracker FX 3.6のサーバ設定ファイルは利用できますか？

      利用できます。

       

    • TimeTracker FX 3.6でエクスポートしたオフラインファイルは利用できますか？

      利用できます。

       

    • バージョンの確認方法を教えてください

      バージョンの確認方法はこちらをご覧ください。

       

 

インストールについて

    • TimeTracker FX 3.7へのバージョンアップ方法を教えてください

      TimeTracker FX 3.6からTimeTracker FX 3.7へのバージョンアップ方法には、以下の2通りがあります。

      • Administratorをサーバマシンで使用する場合

        アップグレード手順は、［セットアップガイド］の ［アップグレードする］をご覧ください。

         

      • Administratorをクライアントマシンからのみ使用する（サーバマシンでは使用しない）場合

        アップグレード手順は、［セットアップガイド］の ［簡単な方法でアップグレードする］をご覧ください。

         

    • Excel連携アドインを後から追加でインストールできますか

      Excel連携アドインは、TimeTracker FX 3.7をインストール後に追加でインストールできます。

      Excel連携アドインを後からインストールする手順はこちらをご覧ください。

【機能について】

  • Web APIとは何ですか？

    TimeTracker FXと他システムのデータ連携をサポートするためのAPIです。

     

  • Web Editionとは何ですか？

    Web アプリケーション版のTimeTracker FXです。
    タイムシートの工数入力機能がブラウザから利用できます。
