TimeTracker V1.xからTimeTracker FXにアップグレードするには、以下の情報が必要です。

アップグレード作業を実施する前に、以下の情報を整理することをお勧めします。

 

サーバマシン

TimeTracker FXのインストール時に利用する情報です。

必要な情報

例

値

サーバマシン名

DCINC-SVR

 

管理者ログイン名

Administrator

 

IPアドレス

192.168.1.1

 

  • TimeTracker FXのインストールには、管理者権限が必要です。

  • IPアドレスは通常利用しません。問題発生時に利用します。

 

SQL Server

［データベース作成ウィザード］で利用する情報です。

必要な情報

例

値

インスタンスの設定

既定　・　○ユーザ指定値

既定　・　ユーザ指定値

インスタンス名

TIMETRACKER

 

サーバ名

DCINC-SVR\TIMETRACKER

 

管理者ログイン名

sa

 

  • インスタンス名は、［既定のインスタンス］の場合は不要です。

  • サーバ名は、［サーバマシン名 + \ + インスタンス名］です。

 

移行元データベース（TimeTracker V1.x）

［データベース作成ウィザード］で利用する、TimeTracker V1.xのデータベースに関する情報です。

  • TimeTracker V1.xインストール時の情報が必要です。

必要な情報

例

値

データベース名

TimeTracker

 

ログイン名

t2user

 

パスワード

t2user

 

 

移行先データベース（TimeTracker FX）

Server Managerで利用する、TimeTracker FXのデータベースに関する情報です。

  • Server ManagerはTimeTracker FX 2.5以降で、TimeTracker FXのデータベースを簡単に管理するためのツールです。
    Server Managerの詳細については、［Server Manager ヘルプ］をご覧ください。

  • TimeTracker FXのデータベースに関する情報を新たに設定します。重要な情報ですので記録・保管してください。

必要な情報

例

値

データベース名

TimeTrackerFX

 

ログイン名

fxuser

 

パスワード

fxuser

 

  • TimeTracker V1.xのデータベースと同じ値は設定できません。
