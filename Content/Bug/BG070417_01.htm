• ［EPSON社製のプリンタで印刷すると、WindowsでSTOPエラーが発生する］　（4/17）を更新した内容です。

 

問題の発生するバージョン

TimeTracker FX 2.0.0以降

 

問題が修正されるバージョン

マイクロソフト社の修正プログラムをご利用ください。

 

内容

以下の環境で計画テーブル（ガントチャート）を印刷すると、STOP 0x0000007F エラーが発生します。

  • Windows XP SP2

  • MS07-017を適用

    以下の手順でMS07-017の有無を確認できます。

    1. ［コントロールパネル］-［プログラムの追加と削除］を開きます。

    2. ［更新プログラムの表示］をチェックします。

    3. Windows XP セキュリティ更新 (KB925902)の有無を確認します。
      存在する場合は、MS07-017が適用されています。

  • EPSON社製プリンタ他

 

印刷できる機能と、エラーが発生する機能は次の通りです。

印刷機能

状況

タイムシートの［個人進捗］ビュー　

印刷できます

プランナーの［計画］ビュー

（計画テーブルおよび［ガントチャート］ペイン）

エラーが発生します

アナリストの［プロジェクトサマリ］ビュー

印刷できます

アナリストの［ガントチャート］ビュー

エラーが発生します

アナリストの［マイルストーンサマリ］ビュー

印刷できます

アナリストの［リソースサマリ］ビュー

印刷できます

 

以下のプリンタで現象が発生することを確認しています。

  • EPSON LP-9400

  • EPSON LP-8600FX

  • EPSON LP-8400FX

  • EPSON LP-1900

  • 記載以外のプリンタでも発生する可能性があります。

 

回避方法

マイクロソフト社より修正プログラムが公開されております。こちらを適用してください。

  • Update for Windows XP (KB935843)

 

  • リンクをクリックすると、当社管理外（マイクロソフト社）のWebサイトを開きます。

  • リンク先が見つからない場合は、マイクロソフト社のダウンロードセンターからダウンロードしてください。

  • 本修正プログラムを適用したことによる問題については、一切の責任を負いかねますのでご了承ください。
