Web Editionのログイン画面が表示されません

更新日:

2014/7/1

対象バージョン:

3.7

TimeTracker FX Web Editionのログイン画面にアクセスする際にエラーが発生し、ログイン画面が表示できない場合があります。
この時の対処方法を紹介します。

症状

TimeTracker FX Web Editionのログイン画面が表示されず、以下のようなエラー画面が表示されます。
エラーの内容によって、画面の表示は異なります。

本現象は以下の場面で発生します。(必ず発生するわけではありません)

  • Web サーバにTimeTracker FX Web Editionをインストールして、クライアントPCからWeb Editionにアクセスする場合

原因と対策

本現象が発生する主な原因を列挙します。
これらの内容が該当しないかご確認ください。

  1. Web Editionのインストール漏れ
  2. Web.configの編集内容に誤りがある
  3. SSL通信を行うためのサーバー証明書が不足している
  4. SSL通信の設定ができていない

これらの原因が該当しない場合は、ご利用中のサーバ・クライアントの環境(OSやIISの種類、ネットワーク環境など)を含む、お問い合わせに必要な情報をこちらまでご連絡ください。

1. Web Editionのインストール漏れ

原因 対策
インストーラー内にあるWeb Edition プログラムフォルダ内のファイルが一つでも欠けた場合、Web Editionを正常に利用できません。 インストールCD内にあるWeb Edition プログラムフォルダをフォルダごとコピーして、すべてのファイルを上書きコピーしてください。

 

2. Web.configの編集内容に誤りがある

原因 対策
Web.configは手動で編集するため、追記するべき個所が異なったり、タイプミスによる入力間違いが起こりやすく、間違って編集されたWeb.configでは、Web Editionを正常に利用できません。 Web.configの設定内容を確認してください。
「Web.config編集ツール」を使用すると、元のWeb.configファイルから確実に更新できます。

 

3. SSL通信を行うためのサーバー証明書が不足している

原因 対策
SSL通信に使用するサーバ証明書について、ブラウザが保持するサーバ証明書に不足があるため、SSL通信自体ができない。 SSL通信に必要な証明書の入手方法とブラウザへのインストール方法について、ネットワーク管理者様にご相談ください。

 

4. SSL通信の設定ができていない

原因 対策
IISで行うSSL通信の設定に漏れがあるため、SSL通信ができずにサイトにアクセスできない。 TimeTracker FX Web Editionセットアップガイドのインストール手順を確認し、SSL通信に必要な設定の漏れが無いか確認してください。
