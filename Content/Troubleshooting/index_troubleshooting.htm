トラブルシューティングについて

お問い合わせの多い問題について、情報を公開いたします。

 

環境

  • 画面の表示が崩れてしまう

  • Web APIが利用できない

  • Web Editionへのログインに失敗します

  • Web Editionのログイン画面が表示されません

  • SQL Server Expressのインストール時にSQL Server Native Clientのインストールに失敗する

  • ［Active Directory連携］機能を有効に設定した環境で、TimeTracker FXにログインできない

  • クライアントPCから共有フォルダが参照できない

  • TimeTracker FXが操作できなくなる

  • TimeTracker FX起動時に、システムエラーが発生する

  • TimeTracker FXがデータベースに接続できません

  • Excel連携アドインをアンインストールしたら、ツールバーが残りました

  • SQL Server Expressのインストール時にエラーが発生する場合がある

  • オートメーションAPIを実行すると、CreateObjectにてエラーが発生する場合がある

 

ダッシュボード

  • ダッシュボードやアナリストのデータが表示されません

 

アナリスト

  • ダッシュボードやアナリストのデータが表示されません

 

プランナー

  • クイックレポートで出力したファイルを開く際にエラーが発生し、レポートが正常に表示されません

  • ［プロジェクトを開く］ダイアログにプロジェクトが表示されません

  • 誰もプロジェクトを編集していないのに、編集中のプロジェクトがある

  • プランナーやタイムシートの内容を保存すると、システムエラーが発生する

 

タイムシート

  • クイックレポートで出力したファイルを開く際にエラーが発生し、レポートが正常に表示されません

  • サイボウズの認証テストが失敗する

  • タスクパッケージからの実績入力に失敗することがある

  • Notesから予定のインポートに失敗する

  • タスクツリーからガントチャート表示を設定するとエラーが発生する場合がある

  • タスクツリーが表示されません

  • プランナーやタイムシートの内容を保存すると、システムエラーが発生する

  • タイムシートを開く際にシステムエラーが発生する
