内容

SQL Server 2005で、データベースを格納する既定のフォルダを、ルートフォルダ（例えば C:\ ）に設定すると、Server Managerでサンプルデータを含むデータベースの作成に失敗する。

  1. SQL Server Management StudioまたはSQL Server Management Studio Expressで既定フォルダをルートフォルダに指定する。

  2. Server Managerで［データベースを新規に作成またはアップグレードする］を選択する。

  3. データベース作成/更新ウィザードで［データベースを新規に作成する］を選択する。

  4. ［サンプルデータを含むデータベースを作成する場合はここをチェックします。］をチェックする。

  5. データベース作成中に、エラーが発生する。

 

回避方法

SQL Server Management StudioまたはSQL Server Management Studio Expressで既定フォルダをルートフォルダ以外に設定してください。
