TimeTracker FX 2.0.2

 

内容

SQL Server プログラムファイルのインストール先と、データファイルの位置が異なる場合、サンプルデータ作成中にエラーが発生します。

 

回避方法

エラーダイアログに表示されたパスの、Dataまでのフォルダを作成してください。

  • ［C:\Program Files\Microsoft SQL Server\MSSQL\Data\TimeTrackerFXSamplebak］の場合は、［C:\Program Files\Microsoft SQL Server\MSSQL］を作成します。
