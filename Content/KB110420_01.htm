【TimeTracker FX 3をお使いの場合】

バージョンアップについて

    • TimeTracker FX 3.1へバージョンアップする必要はありますか？

      TimeTracker FX 3.1に対応するOSについては、こちらをご覧ください。

       

      本バージョンでは、ユーザ様の操作性向上のために機能追加や改善を行っております。

      可能な限りバージョンアップすることを推奨します。

       

    • TimeTracker FX 3.1を入手した後は、どうすればよいですか？

      ご入手いただいた方法によって、インストーラを起動するまでの手順が若干異なります。

      該当する入手方法の手順に従い、インストーラを起動してください。

       

      ダウンロードでTimeTracker FX 3.1をご入手された方

      1. ダウンロードしたファイルをご自身のPC上で展開します。

      2. 展開したフォルダの中にあるautorun.exeを実行してください。

      3. オートランの画面が表示されます。

         

      CDでTimeTracker FX 3.1をご入手された方

      1. CDをご自身のPCのCDドライブに入れます。

      2. しばらくお待ちいただくと、オートランの画面が表示されます。

      3. オートランの画面が表示されない場合は、CDの中にあるautorun.exeを実行してください。

       

    • アップグレードはサーバーマシンもクライアントPCも行う必要がありますか？

      必要ございません。

      FX 3からFX 3.1のアップグレードの場合、サーバーマシンのライセンス情報の更新とクライアントPCのアップグレードするだけで

      ご利用いただけます。サーバマシンとクライアントPCを同時にアップグレードする必要はありません。

      これで、クライアントPCを任意のタイミングでFX 3.1にアップグレードしていただけます。

       

    • TimeTracker FX 3で利用していたライセンスキーをTimeTracker FX 3.1で利用できますか？

      利用できません。

      別途ご提供するTimeTracker FX 3.1のライセンスキーをご利用ください。

       

    • TimeTracker FX 3で利用していたデータベースを引き続き利用できますか？

      TimeTracker FX 3で利用していたデータベースは、ライセンス情報の更新のみで、そのままご利用いただけます。

       

    • TimeTracker FX 3のオートメーションAPIで作成したアプリケーションは利用できますか？

      利用できます。

       

    • TimeTracker FX 3のサーバ設定ファイルは利用できますか？

      利用できます。

       

    • TimeTracker FX 3でエクスポートしたオフラインファイルは利用できますか？

      利用できます。

       

    • バージョンの確認方法を教えてください

      バージョンの確認方法はこちらをご覧ください。

インストールについて

    • TimeTracker FX 3.1へのバージョンアップ方法を教えてください

      アップグレード手順の詳細は、［セットアップガイド］に記載されています。

    • TimeTracker FX 3.1へバージョンアップする場合に気をつけることを教えてください

      バージョンアップの大まかな手順と、その注意事項は以下の通りです。

      1. サーバマシンのTimeTracker FXのバージョンアップ。

      • TimeTracker FX 2.xと、FX 3.1は共存できません。

      • TimeTracker FX 2.xでは、FX 3.1のデータベースを参照できません。
        バージョンアップの際は、ユーザにTimeTracker FXの利用停止をご案内ください。

         

      2. データベースのバージョンアップ。

      • バージョンアップ後は、以前のデータベースに戻すことはできません。

      • バージョンアップの前に、必ずデータベースのバックアップを行ってください。

         

      3. クライアントPCのバージョンアップ。

      • TimeTracker FX 2.xでは、FX 3.1のデータベースを参照できません。

      • 詳細は、［セットアップガイド］の ［TimeTracker FX 2.7以前のバージョンからアップグレードする］をご覧ください。

【機能について】

  • Excel連携アドインとは何ですか？

    TimeTracker FXとExcel間で簡単にデータ同期をできる機能です。

    Excelのアドイン機能として登録されます。

    詳細は、［TimeTracker FX ヘルプ］の ［Excel連携アドイン］に関するトピックをご覧ください。

     

  • Server Managerとは何ですか？

    TimeTracker FXのデータベースを簡単に管理するためのツールです。

    TimeTracker FX 3.5をサーバマシンセットアップでインストールすると、一緒にインストールされます。

    Server Managerの詳細については、 ［Server Manager ヘルプ］をご覧ください。

     

  • サーバ設定ファイル作成ツールがありません

    TimeTracker FX 2.0のサーバ設定ファイル作成ツールの機能は、Server Managerに統合されました。

    詳細は［Server Manager ヘルプ］の［データベースを管理する］－ ［サーバ設定ファイルを作成する］をご覧ください。

     

  • データベース作成ウィザードがありません

    TimeTracker FX 2.0のデータベース作成ウィザードの機能は、Server Managerに統合されました。

    詳細は、［Server Manager ヘルプ］の［データベースを作成または登録する］や ［データベースをアップグレードする］をご覧ください。
