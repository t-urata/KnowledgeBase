プロジェクト情報を編集する

更新日:

2014/1/21

対象バージョン:

3.6

対象エディション:

Professional / Standard

概要

プロジェクトに設定された情報をブラウザ上から編集するツールです。
編集可能なプロジェクトの一覧を、まず表示します。
そこから対象のプロジェクトを指定して、プロジェクト名などの属性情報が編集できます。
続けて他のプロジェクトを編集することも可能です。

使用プログラミング言語

JavaScript

利用方法

本サンプルツールの利用方法は以下のとおりです。

  1. 本ページ下部の［ダウンロード］より［WebAPI_EditProjectData.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. ［WebAPI_プロジェクト情報編集ツール_取扱説明書.pdf］を開きます。
  4. ［WebAPI_プロジェクト情報編集ツール_取扱説明書.pdf］の「2.3 操作方法」の手順に従い操作します。

出力データ

編集対象の候補となるプロジェクトの一覧について、以下の内容をブラウザ上に表示します。

No. フィールド 補足
1 選択 編集対象のプロジェクトを選択するラジオボタンです。
ここで選択したプロジェクトの情報を編集します。
2 プロジェクトコード プロジェクトコードを表示します。
3 プロジェクト名 プロジェクトの名前を表示します。
4 マネージャ名 プロジェクトマネージャの名前を表示します。
5 組織名 プロジェクトマネージャが所属する組織の名前を表示します。
6 開始日 プロジェクトの開始日を「YYYY/MM/DD」の形式で表示します。
7 終了日 プロジェクトの終了日を「YYYY/MM/DD」の形式で表示します。
8 ステータス 終了したプロジェクトは「終了済み」、稼働中のプロジェクトは「稼働中」と表示します。

編集対象プロジェクトの選択後に表示される［プロジェクト情報編集］画面では、以下の項目を表示します。

No. フィールド 補足
1 プロジェクトコード 対象プロジェクトに設定するプロジェクトコードです。
2 プロジェクト名 対象プロジェクトに設定する名前です。
3 マネージャ名 対象プロジェクトを管理するプロジェクトマネージャの名前です。
4 組織

プロジェクトマネージャが所属する組織の名前です。

  • 本項目は［マネージャ名］に指定したアカウントに連動して自動的に変更されます。
5 開始日 対象プロジェクトの開始日です。
6 終了日 対象プロジェクトの終了日です。
7 ステータス 対象プロジェクトのステータスです。
終了したプロジェクトは「終了済み」、稼働中のプロジェクトは「稼働中」を選択します。

注意事項

本サンプルツールでは、ログインユーザの権限によって制限があります。

  • 選択できるプロジェクトは、ログインユーザが編集できるプロジェクトの範囲となります。

なお、サンプルツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。
Web APIの詳細は、TimeTracker FX Web APIガイドをご覧ください。
