TimeSheet.Application.AddActualTimeメソッドを使い、CSVファイルから作業実績をインポートするためのサンプルスクリプトです。

 

利用方法について

本サンプルスクリプトの利用方法を説明します。

  1. ページ末尾の［ダウンロード］より、［AddActualTime.zip］をクリックしてサンプルスクリプトをダウンロードします。

  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。

  3. 同じフォルダに作業実績を入力したCSVファイルを格納します。

  4. TimeTracker FXを起動している場合は、データを保存して終了します。

  5. コマンドプロンプトを起動します。

  6. 以下のコマンドを入力します。

cd サンプルスクリプトを展開したフォルダ

cscript.exe AddActualTime.vbs ログイン名,パスワード,実績ファイル名

  • 実行例は こちらをご覧ください。

    C:\>cd Sample

    C:\Sample>cscript.exe AddActualTime.vbs okamoto,password,impTime.csv

    Microsoft (R) Windows Script Host Version 5.6

    Copyright (C) Microsoft Corporation 1996-2001. All rights reserved.

     

    実績ファイルのインポートを開始します。

     

    実績ファイルをインポートしました。

     

    C:\Sample>

 

インポート可能なファイル形式について

インポートできるファイルは、次のCSV形式です。

  • １行目以降
    実績入力の対象日,開始時刻,終了時刻,タスクID

    • ［実績入力の対象日］･･･作業実績をインポートする日を入力します。

    • ［開始時刻］･･･作業実績の開始時刻を入力します。

    • ［終了時刻］･･･作業実績の終了時刻を入力します。

    • ［タスクID］･･･作業したタスクのタスクIDを入力します

    • タスク名とタスクIDの対応を確認するには、［「割り当てられたタスク」ツリーを取得する］をご利用ください。

 

  • すべての項目は必須です。

  • 作業実績を入力したCSVファイルのサンプルは こちらをご覧ください。

    2008/02/11,9:00:00,12:00:00,300

    2008/02/11,13:00:00,15:00:00,301

    2008/02/11,15:00:00,16:00:00,627

    2008/02/11,16:00:00,18:00:00,134

 

作業実績のインポートに失敗した場合は

プロンプト画面にエラーレコードの情報を出力し、作業実績はインポートされません。

  • 実行例は こちらをご覧ください。

    C:\>cd Sample

    C:\Sample>cscript.exe AddActualTime.vbs okamoto,password,impTime.csv

    Microsoft (R) Windows Script Host Version 5.6

    Copyright (C) Microsoft Corporation 1996-2001. All rights reserved.

     

    実績ファイルのインポートを開始します。

     

    レコード,実績開始日時,タスク名,メッセージ

    2件目,2008/02/11 12:00:00,出張,12:00-13:00は休憩時間です。実績入力できません

    3件目,2008/02/11 12:30:00,会議,12:00-13:00は休憩時間です。実績入力できません

    4件目,2008/02/12 9:45:00,顧客・業者対応,12:00-13:00は休憩時間です。実績入力できません

     

    実績インポートエラー：3件

    実績ファイルを修正し、再度インポートしてください。

     

    C:\Sample>

 

出力する情報について

本サンプルスクリプトは、エラー時に以下の情報を出力します。

フィールド

内容

レコード

エラーが発生したレコードの位置です。

実績開始日時

エラーが発生したレコードの、作業実績の開始日時です。

タスク名

エラーが発生したレコードの、タスク名です。

メッセージ

エラーの内容です。

本サンプルスクリプトを自社の業務にあわせて自由に編集できます。

編集方法についてのお問い合わせは受け付けておりません。オートメーションの詳細は、オートメーションヘルプをご覧ください。
