本ツールは、オートメーションAPIから実行した［実績のエクスポート］のデータから、
［実績のインポート］で利用可能なファイルを作成するサンプルツールです。

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より［CreateImportFile.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. TimeTracker FXを起動している場合は、データを保存して終了します。
  4. ［実績インポート用ファイル作成ツール.xls］を開きます。
  5. 「1.概要～5.本ツールの利用と制約」シートの「2.利用方法」の手順に従い、作業を進めます。

出力する情報について

本サンプルツールは、以下の情報をCSVファイルに出力します。

  • 対象日
  • 開始時刻
  • 終了時刻
  • プロジェクトコード
  • タスクコード
  • メモ
  • 出力したCSVファイルは、［実績のインポート］で使用するCSVファイルのフォーマットになっています。

 

サンプルツールのご利用にあたって

本サンプルツールでは、ログインユーザの[権限範囲]および［実績エクスポート］権限によって制限があります。
詳細については、「1.概要～5.本ツールの利用と制約」シートの「2.利用方法」の「1.利用環境と制限」をご覧ください。

なお、サンプルツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。
Excelマクロに使用するオートメーションの詳細は、オートメーションヘルプをご覧ください。
