Excelマクロで、TimeTracker FXのオートメーション機能を使用し、
［実績のエクスポート］にて出力したファイルに、コスト情報（メンバのコスト単価と実績コスト）を取得し、追加するアドインです。

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より［AddMemberCost.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
    展開されるファイルは以下のとおりです。

    3. ・サンプルツール_コスト情報取得ツール.xla

    …

    ツール本体です。

    ・サンプルツール_コスト情報取得ツール取扱説明書.pdf

    …

    本ツールの取扱説明書です。
  3. 展開したサンプルツール_コスト情報取得ツール取扱説明書.pdfを開きます。
  4. サンプルツール_コスト情報取得ツール取扱説明書.pdfの手順3.2に従い、ツールをセットアップします。
  5. ［実績のエクスポート］にて出力したファイルを開き、ツールを実行します。

サンプルツールの活用について

サンプルツール_コスト情報取得ツール取扱説明書.pdfをご覧ください。
