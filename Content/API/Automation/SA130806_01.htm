「実績工数を誤ったタスクへ入力」や「工数入力するタスクの変更」が発生したとき、それまでタイムシートで入力した工数を一つずつ実績の置き換えをするのは大変な手間ですね。

ここでは実績の置き換え作業を簡単におこなうために、ExcelマクロでTimeTracker FXのオートメーションAPIを使用したサンプルツールをご紹介します。

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より［RreplacedActualTime.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。展開されるファイルは以下のとおりです。
    • サンプルツール_実績工数置き換えツール.xls　…　ツール本体と利用方法を記載したファイルです。
  3. 展開したサンプルツール_実績工数置き換えツール.xlsを開きます。
  4. 利用方法は、シート「1.概要～4.本ツールの利用と制約」を必ずお読みください。

サンプルツールのご利用にあたって

本サンプルツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。
Excelマクロに使用するオートメーションの詳細は、オートメーションヘルプをご覧ください。
