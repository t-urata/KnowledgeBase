Web.configファイルの編集を自動化する

更新日:

2014/6/10

対象バージョン:

3.7

対象エディション:

Professional / Standard

 

Web Editionのセットアップにあたっては、Web.configというファイルの内容を編集する必要があります。
この編集を手作業で行うのは少々煩わしく間違いやすいものです。
そこで、Web.configの編集を自動化するツール「Web.config設定ツール」を紹介します。
このツールを使えば、ボタン一つで簡単に間違いなくWeb.configを編集してくれます。
Web Editionのセットアップ時にぜひご活用ください。

利用方法

本ツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より［EditWebConfig.zip］をクリックして、ツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. ［Web.config設定ツール.xlsm］を開きます。
  4. 「1.概要～3.本ツールの利用と制約」シートの「2.利用方法」の手順に従い、作業を進めます。

注意事項

本ツールは、Excel 2010以降のバージョンに対応しています。
詳細については、「1.概要～3.本ツールの利用と制約」シートの「2.利用方法」の「2.1.利用環境と制限」をご覧ください。

なお、本ツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。
