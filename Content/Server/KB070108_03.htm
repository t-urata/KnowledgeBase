osql ユーティリティを利用して、データベース（MSDE2000）に接続する方法を説明します。

  • osqlユーティリティとは

    • ODBCを使用してデータベースサーバに接続し、Transact-SQL ステートメント、システム プロシージャなどを実行します。

    osqlユーティリティを利用すると、データベースサーバに対して、様々な命令を行えます。

 

操作方法

  1. Windowsの［スタート］メニューから［ファイル名を指定して実行］をクリックします。

  2. 以下のコマンドを入力して［OK］ボタンをクリックします。

cmd

  1. コマンドプロンプトが開きます。

  2. コマンドプロンプトに以下のコマンドを入力し、MSDE2000に接続します。

osql -S サーバ名 -U 管理者権限のあるユーザ -P パスワード

  • サーバ名は、［サーバマシン名 + \ + インスタンス名］です。

 

実行例

以下の環境の場合の実行例です。

  • サーバマシン名･･･TT-SVR

  • SQL Server（MSDE） インスタンス名･･･TimeTracker

  • SQL Server（MSDE） saアカウントのパスワード･･･admin

  • 入力するコマンドの例

C:\>osql -S TT-SVR\TimeTracker -U sa -P admin
