バージョンの確認方法について

TimeTracker FXのバージョンは以下の3つの方法のいずれかで確認できます。

確認方法1

  1. TimeTracker FXを起動します。
  2. [ヘルプ]の[バージョン情報]をクリックします。
  3. [バージョン情報]ダイアログに記載されているバージョンを確認します。

確認方法2

  1. Windowsの[コントロールパネル]を開きます。
  2. [プログラムの追加と削除]をクリックします。
  3. [TimeTracker FX]をクリックします。
  4. [サポート情報を参照するには、ここをクリックしてください。]をクリックします。
  5. [サポート情報]ダイアログに記載されているバージョンを確認します。

確認方法3

  1. Server Managerを起動します。
  2. Server Managerの右上に表示される[TimeTracker FX バージョン]を確認します。
