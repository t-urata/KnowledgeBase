TimeTracker FXは社内LANでの利用を想定して動作性能を保証しております。
クライアントとサーバ間のネットワークにWAN環境を含む場合は、以下の点についてご注意ください。

注意事項

  • TimeTracker FXが問題なく利用できるか確認する
    TimeTracker FXはWAN環境での動作性能を検証しておりません。
    WAN環境を含むネットワークでTimeTracker FXを運用される場合は、お客様ご自身でタイムシートやプランナーの動作をご確認いただき、利用できるか判断してください。
     
  • TimeTracker FXの通信内容を保護するか検討する
    TimeTracker FXの通信はSSL通信などで暗号化しておりません。
    そのため通信内容を傍受・解析される可能性がございます。
    セキュリティ対策が必要な場合はVPNサービスの利用などをご検討ください。
     
  • SQL Serverへの不正アクセス対策が必要か検討する
    WAN環境でTimeTracker FXを利用いただく場合、SQL Serverの通信ポートをWAN環境に対してアクセス可能にする必要がございます。
    そのため、SQL Serverに対して第三者から不正アクセスを受ける可能性がございます。
    セキュリティ対策が必要な場合は、以下のような対策をご検討ください。
    • ファイアウォールで通信を制御する。
    • ポート番号を不正アクセスされにくい（あまり使われない）番号に変更する。
      • SQL Serverが利用するポート番号につきまして、詳細はこちらをご覧ください。
