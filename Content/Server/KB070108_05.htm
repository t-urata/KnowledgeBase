TimeTracker V1.xからTimeTracker FXにアップグレードするには、以下の情報が必要です。

アップグレード作業を実施する前に、以下の情報を整理することをお勧めします。

 

サーバマシン

TimeTracker FXのインストール時などに利用する情報です。

  • 移行元サーバマシン

必要な情報

例

値

サーバマシン名

OLD-SVR

 

管理者ログイン名

Administrator

 

IPアドレス

192.168.1.1

 

  • 移行先サーバマシン

必要な情報

例

値

サーバマシン名

NEW-SVR

 

管理者ログイン名

Administrator

 

IPアドレス

192.168.1.2

 

  • TimeTracker FXのインストールには、管理者権限が必要です。

  • IPアドレスは通常利用しません。問題発生時に利用します。

 

SQL Server（MSDE）

データベースのコピーおよび移行先データベースサーバの設定で利用する情報です。

  • 移行元サーバマシンのデータベースサーバ

必要な情報

例

値

インスタンスの設定

既定　・　○ユーザ指定値

既定　・　ユーザ指定値

インスタンス名

TIMETRACKER

 

サーバ名

OLD-SVR\TIMETRACKER

 

管理者ログイン名

sa

 

  • 移行先サーバマシンのデータベースサーバ

必要な情報

例

値

インスタンスの設定

既定　・　○ユーザ指定値

既定　・　ユーザ指定値

インスタンス名

TIMETRACKER

 

サーバ名

NEW-SVR\TIMETRACKER

 

管理者ログイン名

sa

 

  • インスタンス名は、［既定のインスタンス］の場合は不要です。

  • サーバ名は、［サーバマシン名 + \ + インスタンス名］です。

 

データベース（TimeTracker FX）

データベースのコピーおよび移行先データベースサーバの設定で利用する情報です。

  • 移行元データベース

  • 移行元サーバマシンのTimeTracker FXインストール時の情報が必要です。

必要な情報

例

値

データベース名

TimeTrackerFX

 

ログイン名

fxuser

 

パスワード

fxuserpwd

 

  • 移行先サーバマシンのTimeTracker FXデータベース情報

  • この情報をもとに、［サーバ設定ファイル］を作成します。

  • 移行元データベースと同じ値を設定できます。

必要な情報

例

値

データベース名

TimeTrackerFX

 

ログイン名

fxuser

 

パスワード

fxuserpwd
