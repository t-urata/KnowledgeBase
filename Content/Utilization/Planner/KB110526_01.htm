このトピックの対象

TimeTracker FX 3.1以降

 

内容

TimeTracker FXには、Microsoft Projectと連携してプロジェクトデータを活用するためにいろいろな仕組みが搭載されています。

以下のような使い方が可能です。いずれもMicrosoft Projectがインストールされていなくても実行できます。

 

  1. TimeTracker FXのプロジェクト情報をMicrosoft Projectで確認する

  2. Microsoft Projectのプロジェクト情報をTimeTracker FXで確認する

  3. Microsoft Projectのプロジェクト情報をTimeTracker FXのプロジェクトに取り込む

 

これらの機能を通じて、Microsoft Projectからの移行がスムーズに実現できます。

また、Microsoft Projectユーザーとの情報共有にも活用できます。

 

TimeTracker FXのプロジェクト情報をMicrosoft Projectで確認する

TimeTracker FXのプロジェクトを、Microsoft Projectのプロジェクトファイル形式（XML）で出力します。

エクスポートされたファイルはMicrosoft Projectで直接開き、内容を確認することができます。

TimeTracker FXで作成した計画を、Microsoft Projectを使用している顧客や委託先などの関係者にも簡単に配布し、報告や情報展開ができます。

 

 

TimeTracker FXのプロジェクトを、Microsoft Project形式で出力（エクスポート）方法の詳細は、こちらをご覧ください。

 

 

Microsoft Projectのプロジェクト情報をTimeTracker FXで確認する

Microsoft Projectのプロジェクトファイル（MPPもしくはXML形式）をTimeTracker FXで直接開き、内容を確認することができます。

このビューア機能により、Microsoft Projectがなくても顧客や委託先がMicrosoft Projectで作成した計画を直接開き、閲覧できます。

 

 

TimeTracker FXでMicrosoft Projectのプロジェクトファイルを閲覧方法の詳細は、こちらをご覧ください。

 

 

Microsoft Projectのプロジェクト情報をTimeTracker FXのプロジェクトに取り込む

Microsoft ProjectのプロジェクトをTimeTracker FX上のプロジェクトにインポートします。

MPPあるいはXML形式のプロジェクトファイルがインポート可能です。

Microsoft ProjectからTimeTracker FXへプロジェクト管理ツールを切り替える場合でも、既存のデータがスムーズに移行できます。

 

 

Microsoft Projectの計画をTimeTracker FXにインポートする方法の詳細は、こちらをご覧ください。
