このトピックの対象

TimeTracker FX 3.1以降

 

内容

Microsoft Projectからインポート、および、Microsoft Projectへのエクスポートができます。

これにより、Microsoft Projectを使用している顧客の計画をTimeTracker FXで確認したり、

Microsoft Projectを使用している委託先へ、TimeTracker FXで作成した計画を

Microsoft Projectで使用するファイル形式で送付する事ができます。

これで、Microsoft Projectを使用している顧客や委託先とのスケジュール情報の交換がスムーズになります。

 

・Microsoft Projectからインポート：

WBS作成に顧客が作成した計画を利用することもできます。

・MicrosoftProjectへエクスポート：

TimeTracker FXで作成した計画をそのままMicrosoft Projectで使用するファイル形式で出力できます。

・Microsoft Projectファイルを開く：

Microsoft Projectファイルの計画をビューアで簡単に確認できます。

※ビューアの詳細はこちらをご覧ください。

 

Microsoft Projectからインポート

顧客が作成した計画を利用することもできます。

※FX 3までは、Microsoft Projectのインストールが必要でした。

 

  1.  Microsoft Project にてプロジェクトをMPPもしくはXML形式で保存します。

 

  1. [ファイル]メニューから[Microsoft Projectからインポート]を選択します。

[Microsoft Projectからインポート]ダイアログで、Microsoft Projectファイルと、インポート項目を選択します。

 

 

Microsoft Projectへエクスポート

TimeTracker FXのプロジェクトをMicrosoft Projectで閲覧することができます。

Microsoft Projectをお使いの取引先や関係者との間で、プロジェクトの計画や進捗情報を共有するのに大変便利です。

 

  1.  [ファイル]メニューから[Microsoft Projectへエクスポート]を選択します。

[Microsoft Projectへエクスポート]ダイアログで、ファイル名と、エクスポート項目を選択します。

 

  1. Microsoft Projectを起動し、出力されたファイルを開きます。

  • Microsoft Projectで「新しいプロジェクトとしてインポートする」で開いた場合のみ動作を保証します。
