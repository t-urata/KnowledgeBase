このトピックの対象

TimeTracker FX 3.1以降

 

内容

Microsoft Projectをインストールしていなくても、Microsoft Projectのプロジェクトの閲覧ができるビューアを搭載しました。

Microsoft Projectを使用している顧客、委託先との連携性が高まります。

Offline EditionにもMicrosoft Projectのビューアを搭載していますので、外出先で閲覧することもできます。

 

 

Microsoft Project でプロジェクトを保存

Microsoft Project にてプロジェクトをMPPもしくはXML形式で保存します。

 

TimeTracker FXでMicrosoft Projectのプロジェクトを開く

  1.  ダッシュボードのエクスプローラバーから［Microsoft Projectファイルを開く］をクリックします。

 [Micrsoft Projectのファイルを開く]ダイアログにて閲覧対象のMicrosoft Projectのファイルを選択します。

 

 

  1. TimeTrackerFX上にプロジェクトの計画テーブルとガントチャートが表示されます。

  • 本機能ではプロジェクトの編集はできません。
