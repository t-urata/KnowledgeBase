【以前のTimeTracker FX をお使いの場合】

バージョンアップについて

    • TimeTracker FX 3.8へバージョンアップする必要はありますか？

      本バージョンでは、ユーザ様の操作性向上のために機能追加や改善を行っております。

      可能な限りバージョンアップすることを推奨します。

       

    • TimeTracker FX 3.8を入手した後は、どうすればよいですか？

      ダウンロードファイルを展開した中または、CDの中にあるautorun.exeを実行してください。

       

    • アップグレードはサーバーマシンとクライアントPCそれぞれでどのように行えばよいですか？

      サーバーマシンはTimeTracker FXとデータベースをバージョンアップします。

      クライアントPCはTimeTracker FXをバージョンアップします。

      クライアントPCはTimeTracker FX 3.7、TimeTracker FX 3.6のままでも利用できます。

      TimeTracker FX 3.8の機能を利用する場合はクライアントPCもアップグレードしてください。

      TimeTracker FX 3.5以前をお使いの場合、ログインするとエラー画面が表示され、ログインできません。

      TimeTracker FX 3.5以前からバージョンアップする場合は、サーバマシンとクライアントPCに対して同時期に行ってください。

       

    • 以前のTimeTracker FXで利用していたライセンスキーをTimeTracker FX 3.8で利用できますか？

      利用できません。

      別途ご提供するTimeTracker FX 3.8のライセンスキーをご利用ください。

       

    • 以前のTimeTracker FXで利用していたデータベースを引き続き利用できますか？

      以前のTimeTracker FXで利用していたデータベースは、ライセンス情報の更新のみで、そのままご利用いただけます。

       

    • 以前のTimeTracker FXのオートメーションAPIで作成したアプリケーションは利用できますか？

      利用できます。

       

    • 以前のTimeTracker FXのサーバ設定ファイルは利用できますか？

      利用できます。

       

    • TimeTracker FXでエクスポートしたオフラインファイルは利用できますか？

      利用できます。

       

    • バージョンの確認方法を教えてください

【機能について】

  • Web APIとは何ですか？

    TimeTracker FXと他システムのデータ連携をサポートするためのAPIです。

     

  • Web Editionとは何ですか？

    Web アプリケーション版のTimeTracker FXです。
    タイムシートの工数入力機能がブラウザから利用できます。
