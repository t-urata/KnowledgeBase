このトピックの対象

TimeTracker FX 評価版

TimeTracker FX 3以降の製品版

 

症状

SQL Server Express インストール中に以下のダイアログが表示され、インストールに失敗してしまう。

 

原因

本現象は、SQL Server Native Clientが既にインストールされていることが原因で発生します。

※SQL Server Expressのインストーラは、SQL Server Native Clientのインストールも含めたパッケージとなっているため、

SQL Server Native Clientのインストールに失敗することで、SQL Server Express本体のインストールも失敗します。

 

回避方法

既にインストールされているSQL Server Native Clientをアンイストールすることで、回避することができます。

SQL Server Native Clientをアンイストール後に、再度SQL Server Expressのインストールを実施してください。
