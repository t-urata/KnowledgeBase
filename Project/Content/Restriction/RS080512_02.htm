このトピックの対象

TimeTracker FX 2.5.0以降

 

内容

Windows Vistaでユーザーアカウント制御を有効にした状態の場合、TimeTracker FXのインストール時に表示する［ユーザーアカウント制御］ダイアログをキャンセルすると、エラー画面を表示する。

 

  1. Vistaで、ユーザーアカウント制御を有効にする。

  2. autorunの画面からTimeTracekr FXをインストールする。

  3. 表示される［ユーザーアカウント制御］ダイアログで［キャンセル］をクリックする。

  4. エラーダイアログを表示する。

 

回避方法

回避方法はございません。

エラーが発生することで、お客様の環境に影響を与えませんので、そのまま［OK］をクリックし、インストールを中止してください。
