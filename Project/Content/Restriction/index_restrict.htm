制約事項

製品の制約事項を公開しています。

 

プランナー

  • 誰もプロジェクトを編集していないのに、編集中のプロジェクトがある

  • Microsoft Projectからインポート中に、Microsoft Projectで新規プロジェクトを作成するとインポート処理が停止する

  • Microsoft Projectからのインポートでネットワークドライブ上のファイルを指定すると、エラーが発生する

  • 新規に追加したノードとExcelのテーブルの対応が関連付かない場合がある

 

タイムシート

  • Outlookの定期的な予定の例外アイテムは予定のインポートしない

  • タイムシートのオプションダイアログが正しく表示されない場合がある

 

オートメーション

  • オートメーションで、タイムシートのタスクツリーを読み込むと、ダイアログが表示される場合がある

  • オートメーションで、オフライン状態を適切に判定できない場合がある

 

Server Manager

  • Server Managerで定期バックアップを設定しても、次回実行日時が更新されない場合がある

  • Server Managerのデータベースのプロパティ画面でデータファイルのファイル名が表示できない場合がある

  • Server Managerでデータベースのコピーに失敗する場合がある

 

その他

  • マルチディスプレイ利用時にTimeTracker FXの一部機能が正常に動作しない場合がある

  • ［画面－DPI設定］を「大きなサイズ（120DPI）」にしている場合、ダッシュボードのエクスプローラーバーの幅がだんだん大きくなる

  • ヘルプメニューから製品サイトへアクセスすると、エラーが発生する場合がある

  • Vistaにインストールする場合、ユーザーアカウント制御ダイアログをキャンセルすると、エラー画面を表示する

  • Vistaでautorunのリンクをクリックすると、ブラウザなどが複数起動する場合がある
