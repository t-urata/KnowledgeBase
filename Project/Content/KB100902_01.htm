TimeTracker FX 3 についてご説明します。

 

本バージョンの概要

本バージョンは、Excelの計画書の移行支援や現実的なスケジューリング機能など、計画・進捗管理を大幅に強化しました。

さらに操作性や権限・セキュリティなども向上し、ユーザ様の多くの要望に対応したメジャーバージョンアップです。

 

新たに追加・改善された機能

概要をご確認される方は、こちらをご覧ください。

各機能の詳細をご確認される方は、こちらをご覧ください。

【TimeTracker FX 2.xからのバージョンアップについて】

  • TimeTracker FX 3へバージョンアップする必要はありますか？

    本バージョンでは、ユーザ様の操作性向上のために機能追加や改善を行っております。

    可能な限りバージョンアップすることを推奨します。

     

  • TimeTracker FX 3を入手した後は、どうすればよいですか？

    ご入手いただいた方法によって、インストーラを起動するまでの手順が若干異なります。

    該当する入手方法の手順に従い、インストーラを起動してください。

    • ダウンロードでTimeTracker FX 3をご入手された方

    1. ダウンロードしたファイルをご自身のPC上で展開します。

    2. 展開したフォルダの中にあるautorun.exeを実行してください。

    3. オートランの画面が表示されます。

    • CDでTimeTracker FX 3をご入手された方

    1. CDをご自身のPCのCDドライブに入れます。

    2. しばらくお待ちいただくと、オートランの画面が表示されます。

    3. オートランの画面が表示されない場合は、CDの中にあるautorun.exeを実行してください。

       

  • TimeTracker FX 2.xと、TimeTracker FX 3を混在させて利用できますか？

    利用できません。

    サーバマシンのバージョンと、クライアントPCのバージョンが異なるときにログインすると、エラー画面が表示され、ログインできません。

    サーバマシンのバージョンアップとクライアントPCのバージョンアップは同時期に行ってください。

     

  • TimeTracker FX 2.xで利用していたライセンスキーをTimeTracker FX 3で利用できますか？

    利用できません。

    別途ご提供するTimeTracker FX 3のライセンスキーをご利用ください。

     

  • TimeTracker FX 2.xで利用していたデータベースを引き続き利用できますか？

    TimeTracker FX 2.0/2.5/2.7で利用していたデータベースを、TimeTracker FX 3で利用するデータベースにバージョンアップしてご利用いただけます。

    詳細は、［セットアップガイド］の ［アップグレードする］をご覧ください。

     

  • TimeTracker FX 2.xのオートメーションAPIで作成したアプリケーションは利用できますか？

    利用できます。

     

  • TimeTracker FX 2.xのサーバ設定ファイルは利用できますか？

    利用できます。

     

  • TimeTracker FX 2.xでエクスポートしたオフラインファイルは利用できますか？

    利用できません。

    TimeTracker FX 3にバージョンアップする前に、サーバにデータをインポートしてください。

     

  • バージョンの確認方法を教えてください

    バージョンの確認方法はこちらをご覧ください。

 

【インストールについて】

  • TimeTracker FX 3へのバージョンアップ方法を教えてください

    アップグレード手順の詳細は、［セットアップガイド］に記載されています。

    ［セットアップガイド］の ［アップグレードする］をご覧ください。

     

  • TimeTracker FX 3へバージョンアップする場合に気をつけることを教えてください

    バージョンアップの大まかな手順と、その注意事項は以下の通りです。

    1. サーバマシンのTimeTracker FXのバージョンアップ。

    • TimeTracker FX 2.xと、FX 3は共存できません。

    • TimeTracker FX 2.xでは、FX 3のデータベースを参照できません。
      バージョンアップの際は、ユーザにTimeTracker FXの利用停止をご案内ください。

    1. データベースのバージョンアップ。

    • バージョンアップ後は、以前のデータベースに戻すことはできません。

    • バージョンアップの前に、必ずデータベースのバックアップを行ってください。

    1. クライアントPCのバージョンアップ。

    • TimeTracker FX 2.xでは、FX 3のデータベースを参照できません。

    • 詳細は、［セットアップガイド］の ［アップグレードする］をご覧ください。

       

  • Excel連携アドインを後から追加でインストールできますか

    Excel連携アドインは、TimeTracker FX 2.7/FX 3をインストール後に追加でインストールできます。

    Excel連携アドインを後からインストールする手順は以下の通りです。

    1. TimeTracker FX 2.7/FX 3のインストーラを起動します。

    2. ［プログラムの保守］の画面で、［変更］を選択します。

    3. ［カスタム セットアップ］の画面で、［Excel連携アドイン］を［この機能をローカルのハードディスク ドライブにインストールします。］に設定します。

    4. そのままインストーラに従い、インストールします。

 

【機能について】

  • Excel連携アドインとは何ですか？

    TimeTracker FXとExcel間で簡単にデータ同期をできる機能です。

    Excelのアドイン機能として登録されます。

    詳細は、［TimeTracker FX ヘルプ］の ［Excel連携アドイン］に関するトピックをご覧ください。

     

  • Server Managerとは何ですか？

    TimeTracker FXのデータベースを簡単に管理するためのツールです。

    TimeTracker FX 2.7をサーバマシンセットアップでインストールすると、一緒にインストールされます。

    Server Managerの詳細については、 ［Server Manager ヘルプ］をご覧ください。

     

  • サーバ設定ファイル作成ツールがありません

    TimeTracker FX 2.0のサーバ設定ファイル作成ツールの機能は、Server Managerに統合されました。

    詳細は［Server Manager ヘルプ］の［データベースを管理する］－ ［サーバ設定ファイルを作成する］をご覧ください。

     

  • データベース作成ウィザードがありません

    TimeTracker FX 2.0のデータベース作成ウィザードの機能は、Server Managerに統合されました。

    詳細は、［Server Manager ヘルプ］の［データベースを作成または登録する］や ［データベースをアップグレードする］をご覧ください。
