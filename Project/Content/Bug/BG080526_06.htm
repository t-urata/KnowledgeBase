問題の発生するバージョン

TimeTracker FX 2.5.0

 

問題が修正されるバージョン

未定

 

内容

Vista環境で、Excelを起動したまま、TimeTracker FXをアンインストールすると、Excel連携アドインはインストールできるが、Excelにツールバーが残ったままとなる。

  1. Vista環境に、TimeTracker FX（Excel連携アドインを含む）をインストールする。

  2. Excelを起動する。

  3. TimeTracker FXをアンインストールする。

  4. Excelが起動したままであることを告げるダイアログが表示されるので、Excelを閉じずにそのままアンインストールを継続する。

  5. アンインストール完了後も、ExcelにExcel連携アドインのツールバーが表示されたままとなる。

 

回避方法

TimeTracker FXをExcel連携アドインを含め、再インストールし、Excelを終了した状態で、再度アンインストールしてください。
