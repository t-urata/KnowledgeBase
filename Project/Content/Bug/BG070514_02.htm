問題の発生するバージョン

TimeTracker FX 2.0.0以降

 

問題が修正されるバージョン

TimeTracker FX 2.5.0

 

内容

以下の操作を行うとシステムエラーが発生します。

 

【手順】

  1. タイムシートを開きます。

  2. 任意のタスクを［マイタスク］に登録します。

  3. 登録したタスクをクリックして、表示名を編集できる状態にします。

  4. 編集できる状態で、タスクアイコンを右クリックして［削除］をクリックします。

  5. タスクアイコンが消え表示名が残っている状態で、他の操作を行うとシステムエラーが発生します。

 

回避方法

［内容］に記載した手順を避けてください。

本問題が発生した場合は、以下の手順で操作します。

 

【手順】

  1. ［システムエラー］ダイアログの［OK］ボタンをクリックします。

  2. タイムシートを閉じます。

  3. 再びシステムエラーが発生するので、［OK］ボタンをクリックします。

  4. タイムシートを開くと、対象のマイタスクが削除されシステムエラーは発生しません。
