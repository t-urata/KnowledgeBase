問題の発生するバージョン

TimeTracker FX 2.5.0

 

問題が修正されるバージョン

未定

 

内容

Windows Vistaで、ユーザーアクセス制御を有効にした状態で、[プログラムの追加と削除]からExcel連携アドインのインストール構成を変更すると、Excelのツールバーに変更後の内容が反映されない。

  1. Vistaで、ユーザーアクセス制御を有効にする。

  2. TimeTracekr FXをExcel連携アドインも含めてインストールする。

  3. コントロールパネルの［プログラムの追加と削除］から、TimeTracker FXを選択し、［変更］をクリックする。

  4. ［プログラムの保守］画面で［変更］を選択し、Excel連携アドインのみ削除する。

  5. Excelを起動する。

  6. Excelのツールバーに、［TimeTracker FX］ツールバーを表示したままとなっている。

 

回避方法

コントロールパネルの［プログラムの追加と削除］からは変更せずに、TimeTrackerFXSetup.exeを実行し、［プログラムの保守］画面でExcel連携アドインの構成を変更してください。
