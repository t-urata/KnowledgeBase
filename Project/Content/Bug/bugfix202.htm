TimeTracker FX 2.0.2で修正された問題点や不具合です。

 

データベース作成ウィザード

  • データベース作成ウィザードに誤記がある

 

プランナー

  • Microsoft Projectインポートで、達成率のあるタスクの進捗率が0%になる場合がある

  • 計算式のエラーを修正しても、プロジェクトを開きなおすと警告が表示される場合がある

  • Windows2000でMicrosoft Projectをインポートするとエラーになる

  • ノードを指定してプロジェクトを開くと、他プロジェクトを合計したリソース負荷の値が正しくない

 

タイムシート

  • タイムシートのオフラインファイルを、タイムゾーンの異なる端末で開くとシステムエラーが発生する場合がある

  • Outlookから予定をインポートすると、Outlookでメールが開けないなどの問題が発生する場合がある

  • Outlookから予定をインポートすると、時間の間隔で設定した時刻以外に実績が入力できる

 

全般・その他

  • V1シリーズからアップグレードすると、いつまでも終了しない場合がある

  • SQL Server 2005の互換性レベルの設定でリソース負荷の集計に失敗する場合がある
