既知の問題について

当社にて認識している問題点や不具合を公開しています。

 

プランナー

  • Microsoft Projectファイルを開く際に警告ダイアログを表示する場合がある

  • リソース負荷ビューにプロジェクト期間外の計画工数が表示されない場合がある

  • オートフィルタで表示を絞り込むとシステムエラーが発生する

  • [アカウント名]に全角スペースを使用している場合、Excelなどの計画書からWBS構造を自動作成するとリソースが設定されない

  • [日付]フィールドの列の表示幅によって入力した文字列の１文字目が認識されない場合がある

  • プランナーのフィルタなど、一部の機能の条件式でDAY関数が正常に機能しない

  • プロジェクトの権限の設定で設定内容が反映されない場合がある

  • ガントチャートの印刷プレビューでメニューなどが表示されない場合がある

  • プランナーのキーワードフィルタに［"］を使用すると、システムエラーになる

  • グループの設定ダイアログを編集中に、他のアプリケーションを選択すると編集内容が消える

  • ガントバー書式で特定の条件式を設定すると、ガントチャートが赤いバツになる

  • ガントバー書式をクイックモードで編集し、エラーが発生すると条件付書式がすべて削除される

  • キーボードを使用してフィールド値の決定方法を変更すると、エラーになる場合がある

 

タイムシート

  • タイムシートを開く際に、DB接続が失敗すると終了できなくなる

  • タイムシートのオプション画面で休憩時間を設定しても反映されない場合がある

  • 実績レポートをPDF形式でエクスポートすると出力されない箇所がある

  • 実績のインポートで既に開いているファイルを指定するとシステムエラーになる

  • レコーダーでマイタスクに登録したタスクの実績を記録すると、実績のメモが設定されない

  • オフラインファイルのインポートダイアログで、不正なファイル名を指定後、ファイル選択ダイアログを表示するとシステムエラーになる

  • オフラインファイルをエクスポートするとき、システムエラーが発生する場合がある

 

アナリスト

  • アナリストのガントバー書式定義［デフォルト(クイック)］の条件付書式に不正値が設定されている

  • アナリストの初期状態のガントテーブル書式に、条件式がない

 

ダッシュボード

  • 推移モニタの対象ノード選択画面で、ノードがアウトライン番号順に並ばない場合がある

 

ピボット分析

  • ピボット分析にて、フィールド値を持たない（空欄）ノードは、ピボットグラフが表示されない場合がある

  • ピボット分析の「プロジェクト」ならびに「タスク」テーブルで実績コストが正しく集計されない

 

Excel連携アドイン

  • Excel連携アドインで256文字以上の文字列が正しくエクスポートされない

 

オートメーション

  • Vista以降のOSでオートメーションAPIでAdministratorを起動するとエラーが発生する

 

Administrator

  • アカウント一覧で、コスト単価が実際の値に関わらず「0」と表示される場合がある

 

Server Manager

  • バックアップしたデータベースが復元できない場合がある

  • データベースのファイル名が原因で、データベースの操作ができない場合がある

  • SQL Server 2008以降で定期バックアップを実行できない

 

Web API

  • Web APIでプロジェクトマネージャを設定すると、サーバタスクの実行に失敗する場合がある

 

全般・その他

  • 実績工数がリソース負荷の実績データに正しく反映されない

  • Windows Vista/7 で使えるようになったサロゲートペア（4バイト文字）を使うと文字化けする場合がある

  • TimeTracker FXをインストールしたのに、スタートメニューに表示されない場合がある

  • VistaでExcel連携アドインのインストール構成を変更しても、反映されない場合がある

  • Vistaで、Excelを起動したままTimeTracker FXをアンインストールすると、Excelにツールバーが残ったままとなる

  • Windows XPでTimeTracker FXをアンインストールしても、Excel 2007の［アドイン］リボンに無効なExcel連携アドインのボタンが表示される場合がある

  • VistaでTimeTracker FXをアンインストールしても、Excel 2003のツールバーにExcel連携アドインのボタンの表示が残る場合がある
