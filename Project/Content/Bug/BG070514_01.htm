問題の発生するバージョン

TimeTracker FX 2.0.0以降

 

問題が修正されるバージョン

未定

 

内容

Microsoft Project 2000および2002で、多数のリンクを設定したタスクをインポートするとエラーが発生します。

  • Microsoft Project 2003では発生しません。

 

【手順】

  1. Microsoft Projectで、特定のタスクを先行ノードとして十数のリンクを設定して保存します。

  2. TimeTracker FXを起動して、任意のプロジェクトを開きます。

  3. プランナーの［ファイル］メニューから［Microsoft Projectからインポート］をクリックします。

  4. Microsoft Projectのファイルを選択します。

  5. インポート処理中に ［エラー］ダイアログが開きます。

 

回避方法

以下のいずれかの方法があります。

  • リンクはインポートせず、TimeTracker FXでリンクを再設定する方法

［Microsoft Project ファイルのインポート］ダイアログで［タスクのリンク］のチェックをはずします。

  • Microsoft Project 2003を利用する方法

Microsoft Project 2003では、本問題は発生せずインポートできます。
