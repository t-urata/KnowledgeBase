問題の発生するバージョン

TimeTracker FX 2.5.0

 

問題が修正されるバージョン

未定

 

内容

タイムシートのオプション画面で、休憩時間を複数設定できますが、休憩時間を設定後、［OK］ボタンのショートカットで設定画面を終了すると、最後に修正した内容が反映されません。

  1. タイムシートで［オプション］ダイアログを表示します。

  2. ［休憩時間の設定］の［追加］をクリックします。

  3. ［開始時刻］を「02:00」に設定します。

  4. ［終了時刻］を「02:45」に設定します。

  5. ［Alt］を押しながら、［O］を押すと、入力内容が反映されず、エラーダイアログを表示します。

 

回避方法

マウスで［OK］ボタンをクリックしてください。
