毎日の実績入力を簡単にするテクニックをご紹介します。

 

範囲を指定して工数を入力する

あらかじめ範囲を指定しておき、その時間に作業実績を入力できます。

  1. ［タイムシート］ペインで、作業実績を入力したい時間帯を選択します。

  2. ［タスクツリー］ペインで割り当てるタスクをクリックします。

  3. クリックしたタスクを、［タイムシート］ペインの選択した時間帯へドラッグ&ドロップします。

  4. ドロップしたタスクの開始時刻と終了時刻は、選択した時間帯に合わせて登録されます。

 

 

オートフィット機能を利用する

登録済みの作業実績の開始時刻や終了時刻と重なる時間帯を選択しても、入力した実績は登録済みの作業実績と重複しません。

  1. ［タイムシート］ペインで、登録済みの作業実績の開始時刻や終了時刻と重なる時間帯を選択します。

  2. ［タスクツリー］ペインで割り当てるタスクをクリックします。

  3. クリックしたタスクを、［タイムシート］ペインの選択中の時間帯へドラッグ&ドロップします。

  4. ドロップしたタスクの開始時刻と終了時刻は、登録済みの作業実績と重複しないように登録されます。

 

 

オートスプリット機能を利用する

選択した時間帯に登録済みの実績がある場合は、登録済みの作業実績の開始時刻前の実績と終了時刻後の実績に分割します。

  1. ［タイムシート］ペインで、登録済みの作業実績を含む時間帯を選択します。

  2. ［タスクツリー］ペインで割り当てるタスクをクリックします。

  3. クリックしたタスクを、［タイムシート］ペインの選択中の時間帯へドラッグ&ドロップします。

  4. ドロップしたタスクは、登録済みの作業実績と重複しないように分割して登録されます。
