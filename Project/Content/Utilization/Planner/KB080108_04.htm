工数指標（TPI）は、生産性を表します。

 

説明

現時点の進捗を、計画工数に対して何倍の生産性で進んでいるかを表します。

TPIの値

内容

1.0より小さい

計画より工数を多く使っています。

1.0と等しい

計画通りです。

1.0より大きい

計画より工数を使っていません。

 

計算式

工数指標（TPI）は、以下の計算式で算出します。

工数指標（TPI） = ［実績獲得工数］ / ［実績工数］

  • ［実績獲得工数］ = ［計画工数］ * ［実績進捗率］

    ［実績進捗率］ = ［進捗率］

    進捗率の決定方法により、計算方法が異なります。

  • ［実績工数］ = タイムシートで入力した作業実績時間

 

工数指標（TPI）を使う

工数指標（TPI）を使ってプロジェクトの進捗を管理します。

  1. 以下のフィールドを適切に設定します。

  • ［進捗率］

  • ［計画工数］と［実績工数］

  1. 工数指標（TPI）の値の推移をモニタリングします。

  2. 必要に応じて計画工数の変更などの対応を行います。

  • TPIの値が1.0から大きく異なる場合は、計画工数を見直します。

  • ［最終予測工数］は、計画工数を見直す参考値として活用できます。
