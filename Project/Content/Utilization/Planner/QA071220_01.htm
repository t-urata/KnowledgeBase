加重平均とは、平均値を計算する際に、各数値の重要度に応じた係数を乗じて重み付けする計算方法です。
TimeTracker FXでは、計画工数を係数として利用します。

単純平均と加重平均

単純平均と加重平均の計算式を説明します。

  • 単純平均

（ノードの進捗率 +ノードの進捗率 + ノードの進捗率） / ノード数の和

  • 加重平均

（ノードの進捗率 * 係数 + ノードの進捗率 * 係数 + ノードの進捗率 * 係数 ） / 係数の和

係数（%） = ノードの計画工数 / ノードの計画工数の合計 * 100

加重平均の計算例

簡単な計算例を紹介します。

  1. タスクパッケージXには、以下の状況のタスク1～タスク3があります。

  2. 係数を計算します。

    3. 係数 = 各タスクの計画工数 / タスクの計画工数の合計

     

    計算式と結果

    タスク1

    200H / 1000H = 20%

    タスク2

    300H / 1000H = 30%

    タスク3

    500H / 1000H = 50%
  3. 各タスクの進捗率に、係数を乗じた平均を求めます。

    4. （60% * 20% + 40% * 30% + 20% * 50%） / 100% = 34%
  4. 単純平均と加重平均の結果は、それぞれ次の通りです。

    5. 単純平均

    加重平均

    60%

    34%

    加重平均の方が、タスクパッケージX全体の進捗率をより適切に表しています。
