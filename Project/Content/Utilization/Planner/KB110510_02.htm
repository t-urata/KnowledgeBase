このトピックの対象

TimeTracker FX 3.1以降

内容

TimeTracker FXで作成した計画をMicrosoft Projectへエクスポートできます。
Microsoft Projectを使っている他部署や社外の方とデータをやり取りする場合、計画や進捗のデータのやり取りがネックになりかねません。

しかし、そんな時でも大丈夫です。
TimeTracker FXのプロジェクト計画をMicrosoft Projectが読み込み可能なファイル形式(XMLファイル）でエクスポートできます。

Microsoft Projectへ計画をエクスポートする手順

  1.  [ファイル]メニューから[Microsoft Projectへエクスポート]を選択します。

    2. [Microsoft Projectへエクスポート]ダイアログで、ファイル名と、エクスポート項目を選択します。

  2. Microsoft Projectを起動し、出力されたファイルを開きます。
