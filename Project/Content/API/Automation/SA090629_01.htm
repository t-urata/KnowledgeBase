本ツールは、［メモ］欄の改行コードを削除してCSV形式のファイルに実績をエクスポートするサンプルツールです。

内容

［メモ］欄に改行コードを含む実績を対象に、CSV形式にて［実績のエクスポート］を実行すると
［メモ］欄の改行コードが認識され、正常なフォーマットにて実績情報を出力することができません。

本サンプルツールでは、オートメーションAPIから［実績のエクスポート］を実行し、
取得した実績情報から［メモ］欄の改行コードを削除して、CSV形式のファイルに出力いたします。
［メモ］欄に改行コードを含む実績をCSV形式にてエクスポートする際にご利用ください。

利用方法について

本サンプルツールの利用方法を説明します。

  1. ページ末尾の［ダウンロード］より、［DeleteCRLF.zip］をクリックしてサンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. TimeTracker FXを起動している場合は、データを保存して終了します。
  4. ［実績エクスポート(メモの改行削除)ツール.xls］を開きます。
  5. ［実績エクスポート(メモの改行削除)ツール.xls］の「1.概要～5.本ツールの利用と規約」シートの「2.利用方法」の手順に従い、作業を進めます。
