Excelマクロで、TimeTracker FXのオートメーション機能を使用し、［実績のエクスポート］にて出力したファイルに、タスクの［報告］欄の情報を取得し、追加するサンプルツールです。
［報告］欄に入力された任意のキーワードで、実績工数を集計したい場合などにご活用いただけます。

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より［AddReportInfo.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。展開されるファイルは以下のとおりです。
    • サンプルツール_［報告］欄情報取得ツール.xla　…　ツール本体です。
    • サンプルツール_［報告］欄情報取得ツール取扱説明書.pdf　…　本ツールの取扱説明書です。
  3. サンプルツール_［報告］欄情報取得ツール取扱説明書.pdfを開きます。
  4. サンプルツール_［報告］欄情報取得ツール取扱説明書.pdfの手順3.2に従い、ツールをセットアップします。
  5. ［実績のエクスポート］にて出力したファイルを開き、ツールを実行します。

サンプルツールのご利用にあたって

本ツールは、メンバが入力した［報告］欄の情報を実績工数と合わせて確認したい場合やメンバが入力した実績工数を［報告］欄に入力された任意のキーワードなどで集計・分析したい場合にご活用ください。

なお、サンプルツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。
Excelマクロに使用するオートメーションの詳細は、オートメーションヘルプをご覧ください。
