TimeSheet.Application.GetTaskTreeメソッドを使い、ログインユーザの「割り当てられたタスク」ツリー情報を取得するためのサンプルスクリプトです

［サンプルスクリプト - 作業実績をタイムシートにインポートする］の実績ファイル作成時に、タスクIDを入力する補助として使います。

ユーザはこのスクリプトで出力したファイルのタスクIDを参考にし、実績ファイルを作成してください。

 

利用方法について

本サンプルスクリプトの利用方法を説明します。

  1. ページ末尾の［ダウンロード］より、［GetTaskID.zip］をクリックしてサンプルスクリプトをダウンロードします。

  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。

  3. TimeTracker FXを起動している場合は、データを保存して終了します。

  4. コマンドプロンプトを起動します。

  5. 以下のコマンドを入力します。

cd サンプルスクリプトを展開したフォルダ

cscript.exe GetTaskID.vbs ログイン名 パスワード

  1. 表示するダイアログにプロジェクト名を入力します。

  2. 表示するダイアログにタスク名を入力します。

  • 実行例は こちらをご覧ください。

    C:\>cd Sample

    C:\Sample>cscript.exe GetTaskID.vbs okamoto password

サンプルスクリプトの活用について

本サンプルスクリプトを自社の業務にあわせて自由に編集できます。

編集方法についてのお問い合わせは受け付けておりません。オートメーションの詳細は、オートメーションヘルプをご覧ください。
