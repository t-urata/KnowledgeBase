Planner.Application.GetWBSTreeは、タスクパッケージを取得してWBSツリーの構成を取得するメソッドです。

タスクやマイルストーンなどは取得できません。

  • タスクやマイルストーンの情報を取得するには、 Planner.Application.GetWBSNodesをご利用ください。

 

各メソッドと用途は以下の通りです。

メソッド

用途

GetWBSTree

WBSツリーの階層（タスクパッケージ）を取得します。

GetWBSNode

指定した1つのノードを取得します。

GetWBSNodes

指定したノードを頂点とした、複数のノードを取得します。
