オートメーションAPIを利用しデータベースから［割り当てられたタスク］のタスクIDを取得する［サンプルスクリプト -「割り当てられたタスク」のタスクIDを取得する］について解説します。

本解説は［サンプルスクリプト - 「割り当てられたタスク」のタスクIDを取得する］のコードと一緒にご覧ください。

 

説明するメソッド

本解説では、以下のメソッドについて説明します。

  • GetProjects･･･タスクツリーから、プロジェクトの一覧を取得します。

  • GetTaskTree･･･指定されたプロジェクトのタスクツリーの情報（タスクとタスクパッケージの情報一覧）を取得します。

引数に0を渡すことで、フィルタの「割り当てられたタスク」で表示されるタスクを対象にしています。
