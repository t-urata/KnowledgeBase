Planner.Application.GetProjectsメソッドを使い、プランナーからプロジェクト情報の一覧を取得するためのサンプルスクリプトです。

  • 本サンプルスクリプトでプロジェクト情報を取得するには、［計画閲覧］または［計画作成・編集］権限が必要です。

 

利用方法について

本サンプルスクリプトの利用方法を説明します。

  1. ページ末尾の［ダウンロード］より、［GetProjectList.zip］をクリックしてサンプルスクリプトをダウンロードします。

  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。

  3. TimeTracker FXを起動している場合は、データを保存して終了します。

  4. コマンドプロンプトを起動します。

  5. 以下のコマンドを入力します。

cd サンプルスクリプトを展開したフォルダ

cscript.exe GetProjectList.vbs ログイン名 パスワード

プロジェクトコードをCSVファイルに保存するには

以下の手順で、プロジェクトの稼働状況一覧をCSVファイルに保存できます。

  1. 以下のコマンドを入力します。

cd サンプルスクリプトを展開したフォルダ

cscript.exe //NoLogo GetProjectList.vbs ユーザ名 パスワード >ProjectList.csv

  • 実行例は こちらをご覧ください。

    C:\>cd Sample

    C:\Sample>cscript.exe //NoLogo GetProjectList.vbs okamoto >ProjectList.csv

     

    C:\Sample>

  1. サンプルスクリプトを展開したフォルダに、CSVファイルが作成されます。

 

出力する情報について

本サンプルスクリプトは、以下の情報を出力します。

フィールド

内容

ID

プロジェクトの内部IDです。

Code

プロジェクトコードです。

Name

プロジェクト名です。

ManagerID

マネージャのIDです。

ManagerName

マネージャ名です。

SectionID

マネージャが所属する組織のIDです。

SectionName

マネージャが所属する組織名です。

PlannedStartDate

プランナーの［プロジェクトの設定］で定義した開始日です。

PlannedFinishDate

プランナーの［プロジェクトの設定］で定義した終了日です。

Status

プロジェクトのステータスを表示します。

プロジェクト作成後は［稼働中］です。以下の操作を行うと［終了済み］に変更します。

  • Administratorの［プロジェクト一覧］ビューで［終了］をチェックした場合

  • プランナーの［プロジェクトの設定］で［プロジェクトの終了］をチェックした場合

CheckoutInfo

プロジェクトを編集しているメンバの名前を表示します。

  • 誰も編集していない場合は空文字です。

  • 1人以上のメンバが編集している場合は、［部分編集中］と表示します。

 

サンプルスクリプトの活用について

本サンプルスクリプトを自社の業務にあわせて自由に編集できます。

編集方法についてのお問い合わせは受け付けておりません。オートメーションの詳細は、オートメーションヘルプをご覧ください。
