Excelマクロで、TimeTracker FXのオートメーションAPIを使用し、［実績のエクスポート］で出力可能な項目からあらかじめ指定した項目のみを出力するツールです。
本サンプルツールでは、以下のことが可能になります。

  • ［実績のエクスポート］機能の出力内容を独自の構成にする
  • 出力後のデータ編集が不要で、あらかじめ必要な列のみを出力する

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より［CustomizeExportActualTime.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
    展開されるファイルは以下のとおりです。
    • サンプルツール_実績工数データ出力ツール.xls　…　ツール本体と利用方法を記載したファイルです。
  3. 展開したサンプルツール_実績工数データ出力ツール.xlsを開きます。
  4. 操作シートの［操作方法］に従いサンプルツールを実行します。
