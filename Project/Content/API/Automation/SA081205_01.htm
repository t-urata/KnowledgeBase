Excelマクロで、TimeTracker FXのオートメーション機能を使用し、［実績のエクスポート］で出力したファイルに、ノードパスを取得するアドインです。
このアドインでは、タスク単位で出力された実績の末尾に、そのタスクのノードパスを取得し出力します。

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より、［AddNodePaths.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. TimeTracker FXを起動している場合は、データを保存して終了します。
  4. TimeTracker FX データ収集ツールType004説明書.pdfを開きます。
  5. TimeTracker FX データ収集ツールType004説明書.pdfの手順に従い、作業を進めます。

サンプルツールの活用について

TimeTracker FX データ収集ツールType004説明書.pdfをご覧ください。
