Excelマクロで、TimeTracker FXのオートメーション機能を使用し、任意のプロジェクトのタスクに対してリソースごとの情報を取得するアドインです。
このアドインでは、組織一覧データ、プロジェクト一覧データ、リソース一覧データ、WBS一覧データを取得し、新規ワークシートに出力します。
これらのデータは、リソース別の作業一覧や、報告一覧として活用いただけます。

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より、［ReportAssignment.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. TimeTracker FXを起動している場合は、データを保存して終了します。
  4. TimeTracker FX データ収集ツールType002説明書.xlsを開きます。
  5. TimeTracker FX データ収集ツールType002説明書.xlsの手順に従い、作業を進めます。

出力する情報について

ダウンロード後、展開いただいたTimeTracker FX データ収集ツールType002説明書.xlsの［出力データ一覧］シートをご覧ください。
