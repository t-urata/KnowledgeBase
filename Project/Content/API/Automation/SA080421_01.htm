Excelマクロで、TimeTracker FXのオートメーション機能を使用し、ユーザ毎に特定期間の工数を月別に集計する帳票です。

 

利用方法について

本サンプル帳票の利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より、［ ColectActualTime.zip ］をクリックして、サンプル帳票をダウンロードします。

  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。

  3. TimeTracker FXを起動している場合は、データを保存して終了します。

  4. 工数集計サンプル帳票.xlsを開きます。

  5. 工数集計サンプル帳票.xlsのシート「集計」の手順に従い、作業を進めます。

 

出力する情報について

本サンプル帳票は、以下の情報を出力します。

フィールド

内容

アカウントID

TimeTracker FX内でのアカウント管理番号です。

アカウントコード

アカウントコードです。

組織

組織名です。

有効

アカウントの状態が有効か無効かを表示します。

合計実績時間

集計期間の総実績時間です。

月毎の実績時間

集計期間内の月毎に集計した実績時間です。

 

サンプル帳票の活用について

工数集計サンプル帳票.xlsの使い方、ユーザ様のカスタマイズ、注意事項については、工数集計サンプル帳票.xlsのシート「集計」をご覧ください。

 

なお、サンプル帳票のカスタマイズ方法についてのお問い合わせは受け付けておりません。

Excelマクロに使用するオートメーションの詳細は、オートメーションヘルプをご覧ください。
