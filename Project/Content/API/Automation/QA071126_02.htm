Windows環境でオートメーションを呼び出せる言語であれば制約はありませんが、レイトバインドをサポートする言語を推奨しております。

レイトバインドの場合

オートメーションとして公開しているメソッド名などを、そのまま記述できるためプログラミングが容易です。
代表的な開発言語は、Visual BasicやVBA（Visual Basic for Applications）、VBScriptなどです。

アーリーバインドの場合

オートメーションを利用するために、メソッドを別のメソッドの引数として使用するなどの工夫が必要です。
そのため、レイトバインドと比べるとプログラミングが煩雑です。
代表的な開発言語は、Visual C#などです。
