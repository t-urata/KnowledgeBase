Excelマクロで、TimeTracker FXのオートメーション機能を使用し、Excelシートで入力された情報を基に、既存のプロジェクトへタスクを追加するアドインです。

このアドインでは、Excelシートで入力された情報を利用し、複数のプロジェクトに一括でタスクを作成することができます。

 

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より、［AddNodesToProjects.zip］をクリックして、サンプルツールをダウンロードします。

  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。

  3. TimeTracker FXを起動している場合は、データを保存して終了します。

  4. TimeTracker FX データ収集ツールType005説明書.xlsを開きます。

  5. TimeTracker FX データ収集ツールType005説明書.xlsの手順に従い、作業を進めます。

 

サンプルツールの活用について

TimeTracker FX データ収集ツールType005説明書.xlsをご覧ください。
