Excelマクロで、TimeTracker FXのオートメーション機能を使用し、

複数のプロジェクトにプロジェクトメンバを一括で追加するサンプルツールです。

 

本サンプルツールは、新入メンバの配属や組織変更などにより、

複数のプロジェクトに複数のメンバを追加する際にご利用ください。

 

 

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より、［AddProjectMember.zip］をクリックして、サンプルツールをダウンロードします。

  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。

  3. TimeTracker FXを起動している場合は、データを保存して終了します。

  4. ［TimeTracker FX 自動プロジェクトメンバ追加ツール.xls］を開きます。

  5. 「1.概要～5.本ツールの利用と制約」シートの「2.利用方法」の手順に従い、作業を進めます。

 

出力する情報について

ダウンロード後、展開いただいた［TimeTracker FX 自動プロジェクトメンバ追加ツール.xls］の

「1.概要～5.本ツールの利用と制約」シートの「3.出力データ一覧」をご参照ください。

 

サンプルツールのご利用にあたって

本サンプルツールでは、アカウント一覧データとプロジェクト一覧データを取得し、ワークシートへ出力しましす。

情報を取得するために用意されたシートは削除しないでください。


なお、サンプルツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。

Excelマクロに使用するオートメーションの詳細は、オートメーションヘルプをご覧ください。
