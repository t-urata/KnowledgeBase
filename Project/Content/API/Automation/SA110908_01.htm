本ツールは、指定したプロジェクトのタスクにユニークなタスクコードを設定するサンプルツールです。
（複数プロジェクトの指定が可能です）

入力済みの実績を置き換えるためにユニークなタスクコードの設定が必要となった場合にご活用ください。

利用方法について

本サンプルツールの利用方法を説明します。

  1. 本ページ末尾の［ダウンロード］より［SetUniqueTaskCode.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. TimeTracker FXを起動している場合は、データを保存して終了します。
  4. ［タスクコード設定ツール.xls］を開きます。
  5. 「1.概要～4.本ツールの利用と制約」シートの「2.利用方法」の手順に従い、作業を進めます。

サンプルツールのご利用にあたって

本サンプルツールでは、ログインユーザの［計画編集］権限（FX 2.7以前のバージョンでは、［計画作成・編集］権限）
によって制限があります。
詳細については、「1.概要～4.本ツールの利用と制約」シートの「2.利用方法」の「1.利用環境と制限」をご覧ください。

なお、サンプルツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。
Excelマクロに使用するオートメーションの詳細は、オートメーションヘルプをご覧ください。
