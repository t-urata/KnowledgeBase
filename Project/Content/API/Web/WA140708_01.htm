プロジェクトのWBS情報を表示する

更新日:

2014/7/8

対象バージョン:

3.6以降

対象エディション:

Professional / Standard

概要

プロジェクトのWBS情報をブラウザ上に表示するツールです。
閲覧可能なプロジェクトの一覧を、まず表示します。
そこから対象のプロジェクトを指定して、WBS情報を表示できます。
続けて他のプロジェクトのWBSを表示することも可能です。

使用プログラミング言語

JavaScript

利用方法

本サンプルツールの利用方法は以下のとおりです。

  1. 本ページ下部の［ダウンロード］より［WebAPI_GetWBSNodeData.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. ［WebAPI_WBS情報表示ツール_取扱説明書.pdf］を開きます。
  4. ［WebAPI_WBS情報表示ツール_取扱説明書.pdf］の「2.3 操作方法」の手順に従い操作します。

出力データ

WBS情報の表示候補となるプロジェクトを表示する［プロジェクト情報一覧］画面では、以下の項目をブラウザ上に表示します。

No. フィールド 補足
1 選択 WBS情報の表示対象にするプロジェクトを選択するラジオボタンです。
ここで選択したプロジェクトのWBS情報を表示します。
2 プロジェクトコード プロジェクトコードを表示します。
3 プロジェクト名 プロジェクトの名前を表示します。
4 マネージャ名 プロジェクトマネージャの名前を表示します。
5 組織名 プロジェクトマネージャが所属する組織の名前を表示します。
6 開始日 プロジェクトの開始日を「YYYY/MM/DD」の形式で表示します。
7 終了日 プロジェクトの終了日を「YYYY/MM/DD」の形式で表示します。

プロジェクトを選択後に表示される［WBS情報表示］画面では、以下の項目をブラウザ上に表示します。

No. フィールド 補足
1 アウトライン番号 WBSの階層構造を示す番号です。
  • ［ノード種別］がプロジェクトの行は空白になります。
2 ノード名 ノードの名前を表示します。
3 開始日 ノードの開始日を「YYYY/MM/DD」の形式で表示します。
4 終了日 ノードの終了日を「YYYY/MM/DD」の形式で表示します。
5 進捗率 ノードの進捗率を表示します。
  • 進捗率を管理しないノードは「N/A」で表示します。
6 ノード種別 ノードの種類を示す名称です。

注意事項

本サンプルツールでは、ログインユーザの権限によって制限があります。

  • 選択できるプロジェクトは、ログインユーザが閲覧できるプロジェクトの範囲となります。

なお、サンプルツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。
Web APIの詳細は、TimeTracker FX Web APIガイドをご覧ください。
