プロジェクト情報の一覧を収集する

更新日:

2013/12/11

対象バージョン:

3.6

対象エディション:

Professional / Standard

概要

プロジェクトで設定した情報をブラウザ上に一覧表示するツールです。
本ツールでは、指定した条件に一致するプロジェクトの一覧が表示できます。
また、表示したデータはCSVファイルとして出力できます。

使用プログラミング言語

JavaScript

利用方法

本サンプルツールの利用方法は以下のとおりです。

  1. 本ページ下部の［ダウンロード］より［WebAPI_GetProjectList.zip］をクリックして、サンプルツールをダウンロードします。
  2. ダウンロードした圧縮ファイルを、任意のフォルダへ展開します。
  3. ［WebAPI_プロジェクト情報表示ツール_取扱説明書.pdf］を開きます。
  4. ［WebAPI_プロジェクト情報表示ツール_取扱説明書.pdf］の「2.3 操作方法」の手順に従い操作します。

出力データ

以下の内容をブラウザ上に表示します。

No. フィールド 補足
1 No. -
2 プロジェクトコード -
3 プロジェクト名 -
4 組織名 -
5 マネージャ名 -
6 開始日 プロジェクトの開始日の日付を「YYYY/MM/DD」の形式で表示します。
7 終了日 プロジェクトの終了日の日付を「YYYY/MM/DD」の形式で表示します。
8 ステータス 終了したプロジェクトは「終了済み」と表示し、稼働しているプロジェクトは「稼働中」と表示します。

注意事項

本サンプルツールでは、ログインユーザの権限によって制限があります。

  • 選択できるプロジェクトは、ログインユーザが参照できるプロジェクトの範囲となります。

なお、サンプルツールのカスタマイズ方法についてのお問い合わせは受け付けておりません。
Web APIの詳細は、TimeTracker FX Web APIガイドをご覧ください。
