Silent-installing TimeTracker FX

Date:

2016/6/10

Target version:

3.8

Target edition:

Professional / Standard

 

It is very annoying to install TimeTracker FX in many PCs individually on introducing or upgrading.
Here is described how to install in multiple PCs as a batch process.

The merits of this way are:

  • You don't need to deploy the installer to each client PC
  • You don't need to operate on the dialogue; you only need to execute a script file for installing

Overview

You only need to execute the install script on client PCs after storing the installer and the server settings file of TimeTracker FX in a shared folder of the server. Then, installing TimeTracker FX will be automatically completed.

The steps are as follows:

  1. Store the following files in a shared folder of the server to which client PCs can access.
    • Server settings file of TimeTracker FX (TimeTracker.config)
    • Installer of TimeTracker FX (TimeTrackerFXsetup.exe, vcredist_x86.exe)
  2. Execute the install script on client PCs.

With these steps, you will complete installing or upgrading TimeTracker FX in a client PC in a couple of minutes automatically.

Preparing a shared folder

Store the server settings file and the installer of TimeTracker FX in a shared folder of the server to which client PCs can access.

The following example shows that the shared folder (C:\Apps) has been created in the drive C of the server machine ("TTFXServer" as the computer name) where the server settings file and the installer of TimeTracker FX have been stored.

  • "3.7.1.40930" in the image should be replaced with the actual version of TimeTracker FX.
  • The actual version for TimeTracker FX 3.8 is "3.8.0.50320".

Script file

Here is introduced the sample script (SilentInstallScript_en.zip).
Excel Addin will be also installed on installing with this script.

 

The main operations of this sample script are as follows:

  • Saving the location of the server settings file in the registry of a client PC
  • Silent-installing the program of TimeTracker FX
 

You should customize the sample script as follows:

  1. Version of TimeTracker FX
    Actual version of TimeTracker FX to be installed should be applied.
    RowSource code
    16szTimeTrackerFXCurVer = "3.8.0.50320"
 
  1. Location of the server settings file
    The path of your shared folder for the server settings file should be applied.
    RowSource code
    19szTimeTrackerFXConfig = "\\TTFXServer\Apps\TimeTrackerFX\cfg"
 
  1. Location of TimeTracker FX installer
    The pach of your shared folder for the installer should be applied.
    • You must add "\" at the end of the string.
    RowSource code
    22szTimeTrackerFXInstaller = "\\TTFXServer\Apps\TimeTrackerFX\"
 
  1. Specifying the edition of TimeTracker FX to be installed
    You should specify "0" for the Standard Edition or "1" for the Professional Edition.
    RowSource code
    25szInstallType = 1
    • If "0" is specified, TimeTracker FX and Administrator will be installed.
      Server Manager and Excel Addin will not be installed.
    • If "1" is specified, TimeTracker FX, Administrator and Excel Addin will be installed.
      Server Manager will not be installed.
 

Testing the script file

When you have completed preparing shared folder and script file, it is recommended to test installing with the script on several client PCs before actual operation.

  1. Save the script file on a folder and execute it with double-clicking.
    You can't execute it on a server OS or if the latest vesion has already been installed.

 

  1. Installation will automatically be completed in a couple of minutes.
