製品版に必要なデータベース環境

TimeTracker FXを利用するには、データベースサーバが必要です。

 

対応データベースサーバ

  • Microsoft SQL Server

    • 対応するSQL Serverのバージョンはこちら（TimeTracker FXのバージョン別動作環境「FX 3.5」）をご覧ください。

 

必要なSQL Serverの設定

TimeTracker FXをお使いいただく場合、SQL Serverを以下の設定にしてください。

  • 認証モードの設定は、混合モード (Windows 認証と SQL Server 認証)

  • ［既定のインスタンス］と［名前付きインスタンス］のいずれでも動作します。

  • SQL Server 2005以降をご利用の場合は、［互換性レベル］を［SQL Server 2000 (80)］に設定してください。
