サーバマシンに障害が発生した場合に、TimeTracker FXの運用環境を修復するために必要な、バックアップ対象とするデータを説明します。

 

バックアップするデータ

サーバマシンに障害などが発生した場合、TimeTracker FXの運用環境を修復するための情報です。

  • データベース名.mdf

  • データベース名_log.ldf

  • これらのファイルは、［データベース作成ウィザード］または［Server Manager］で作成したデータベースのデータファイルとログファイルです。

  • データベース名は、［データベース作成ウィザード］または［Server Manager］でデータベースを作成時に指定した名前となります。

  • TimeTracker FX 2.0をご利用の方は［バックアップを行う（SQL Server 2005）］をご覧ください。

  • TimeTracker FX 2.5以降をご利用の方は、Server Managerの ［データベースをバックアップする］機能をご利用ください。

  • ［サーバ設定ファイル］
