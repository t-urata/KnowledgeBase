このトピックの対象

SQL Server 2000

 

内容

名前付きインスタンスを利用している場合に、通信ポートを固定する方法を説明します。

  • 既定のインスタンスと名前付きインスタンスが共存している環境では、本手順を行わないでください。

  • 本手順は、既定のインスタンスで動作している場合は必要ありません。

  • 英語のダイアログで説明しますが、日本語の場合も手順は同様です。

 

設定手順

  1. Windowsスタートメニューの［ファイル名を指定して実行］にて［svrnetcn.exe］と入力します。

  2. ［SQL Server Network Utility］ダイアログが開きます。

  3. ［Instance(s) on this server］から、対象のサーバ名（インスタンス）を選択します。

  4. ［Enabled protpcpls］から［TCP/IP］をクリックします。

  5. ［Properties...］ボタンをクリックします。

  6. ［サーバマシン名 - TCP/IP］ダイアログが開きます。

  7. ［Degault Port］に、［1433］と入力します。

  8. 各ダイアログにて、［OK］ボタンをクリックして画面を閉じます。

  9. ［SQL Server サービスマネージャ］にて、SQL Serverを再起動します。
