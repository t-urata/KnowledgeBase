Windows認証でデータベースサーバにログインして、saのパスワードを変更できます。

 

saパスワードの変更方法

  1. Windowsに管理者権限をもったアカウントでログインします。

  2. osqlユーティリティで、MSDEに接続します。

osql -S サーバ名 -E

  1. サーバ名は、［サーバマシン名 + \ + インスタンス名］です。

  2. osqlユーティリティに、以下のコマンドを入力します。

sp_password NULL,'新しいパスワード', 'sa'

go

  • 実行例は、 こちらをご覧ください。

    1> sp_password NULL,'NewPWD', 'sa'

    2> go

    パスワードは変更されました。
