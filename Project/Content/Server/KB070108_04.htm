サーバマシンの移行、サーバタスクが動作に問題があった場合に、サーバタスクを手動で設定する方法を説明します。

  • 本手順は、TimeTracker FX サポートチームより案内があった場合に行ってください。

 

設定方法

  1. TimeTracker FXをインストールしたフォルダの［Config］フォルダを開きます。

通常は［C:\Program Files\Denso Create\TimeTracker FX\Config］です。

  1. フォルダを作成します。フォルダ名の規則は以下の通りです。

［サーバーマシン名 + .（ドット） + インスタンス名 + .（ドット） + データベース名］

  • 作成するフォルダの例

サーバーマシン名･･･FX-SERVER

インスタンス名 ･･･無し（既定のインスタンス）

データベース名 ･･･TimeTrackerFX

作成するフォルダ名 ･･･FX-SERVER..TimeTrackerFX

  1. 上記フォルダに、新しく作成した［サーバ設定ファイル］をコピーします。

  2. Windows［スタート］メニューから、［ファイル名を指定して実行］をクリックします。

  3. services.msc と入力し、［OK］ボタンをクリックします。

  4. ［名前］の一覧から、［TimeTrackerサーバタスク］を探して選択します。

  5. ［TimeTrackerサーバタスク］の［状態］を確認します。

  6. ［状態］が［開始］であれば、右クリックの［再起動］を行います。

［状態］が［開始］以外であれば、右クリックの［開始］を行います。
