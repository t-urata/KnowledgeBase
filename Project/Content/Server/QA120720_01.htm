複数のSQL Serverをインストールして利用できますか?

 

内容

はい。可能です。

SQL Serverでは、インスタンス（SQL Serverの実行単位）を複数作成して各インスタンスで設定を変更したり、バージョンの異なるSQL Serverを1台のサーバ上で実行することができます。

また、1つのインスタンスで複数のデータベースを管理することもできます。

 

 

TimeTracker FXと他アプリケーションを共存させてご運用するには

1台のサーバでTimeTracker FXとその他のSQL Serverを利用するアプリケーションを共存させるには、TimeTracker FXの環境が他のアプリケーションへ影響を与えないように、

専用のインスタンスを作成したうえでの運用を推奨いたします。

 

 

インスタンスの追加方法

インスタンスは、SQL Serverのインストーラから「新規インストール」を実行することで使用したSQL Serverのインストーラのバージョンに応じたインスタンスを追加することができます。
