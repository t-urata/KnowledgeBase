TimeTracker FXでは、データベースサーバに、Microsoft SQL Serverを使用しており、以下の通信ポートを利用します。
SQL Serverの環境により、利用するポートが異なります。

通信ポートについて

  • 既定のインスタンスの場合（SQL Serverを標準の設定でインストールした場合）

1433/TCP

  • 名前付きインスタンスの場合

1434/UDP
任意のTCPポート（1025～65535/TCP）

  • 名前付きインスタンスは動的ポート割り当てプロセスを利用して、OSが利用していない空きポートから、任意のポートを割り当てて待ち受けます。
